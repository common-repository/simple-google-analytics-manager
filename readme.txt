== Description ==

This plugin adds the required javascript for ganalytics.

For more information visit:

[Manager for Google Analytics](https://portalplanet.net/google-analytics-wordpress-plugin/)

== Installation ==

1. Upload or copy the `simple-google-analytics-manager` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the UA from GA (UA-XXXXXXX) to the settings (Admin > Settings > Manager for Google Analytics)
